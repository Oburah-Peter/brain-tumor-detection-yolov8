# Brain Tumor Detection using YOLOv8

This project detects brain tumors from MRI images using the YOLOv8 deep learning model.

## Features

- YOLOv8 object detection
- Brain tumor classification
- Streamlit web application
- MRI image analysis
- AI explainability with heatmaps

## Dataset

MRI Brain Tumor Dataset from Kaggle.

Classes detected:

- Glioma
- Meningioma
- Pituitary tumor

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the Streamlit app:

streamlit run brain_tumor_app.py

## Example Workflow

1. Upload an MRI image
2. The YOLOv8 model detects tumors
3. The system shows bounding boxes and confidence scores
